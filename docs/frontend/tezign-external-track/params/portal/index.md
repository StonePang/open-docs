# Portal业务埋点传参(持续更新中)

## Portal业务中的通用属性

| |  类型 | 说明  | 来源  |
| :------------ | :------------ | :------------ | :------------ |
| portal_id  | string  |  分发看板ID | open_api返回的portalId  |
|  block_id | number  |  内容模块ID | open_api返回的blockId  |
|  content_id | number  | 看板内容ID  |  open_api返回的contentId |
| asset_id  | number  | 素材库父素材ID  | open_api返回的**coreId**  |
